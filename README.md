# Laika 0.1.0 Alpha 3

Private alpha for trusted testers. Thank you for helping!

## Walkthrough

Watch the alpha walkthrough: [LaikaPromo_alpha.mp4](./LaikaPromo_alpha.mp4)

## Install

Download **Laika.dmg** from the
[Laika 0.1.0 Alpha 3 release](https://github.com/Adaptive-Insight/LaikaApp/releases/tag/v0.1.0-alpha.3).
Do not use GitHub's autogenerated “Source code” zip/tar links; those are not
the app installer.

Open the DMG, drag **Laika.app** to `/Applications`, eject the DMG, and launch
Laika from `/Applications`. macOS may ask you to confirm opening the notarized
app on first launch.

## Run

Add or select the folders you want Laika to index. Use **Find** for collection
search, then explicitly select files before asking grounded questions,
comparing sources, summarizing, or extracting facts.

The first model-backed question downloads the promoted model from Hugging Face
(`min-h-wei/Laika`), verifies every file by SHA-256, and caches it locally. The
download is about 2.5 GB, so the first answer can take a few minutes. Later
launches reuse the local cache.

## What’s New in Alpha 3

- Stronger proof boundaries keep visual observations separate from raw document
  and OCR evidence.
- More reliable, restart-safe enrichment with exact source/model freshness
  checks.
- Scoped search filters are applied before rank cutoffs, with clearer match
  explanations.
- Grounded answers preserve subject identity, cross-file conflicts, and exact
  raw-span support more consistently.
- Foreground searches and selected-source questions take priority over
  background enrichment.
- Selected-photo answers use bounded pixel reads and are visibly labeled as
  visual inference.

## Privacy

Model work runs on-device after installation. This alpha has no telemetry,
crash reporting, or automatic update checks. Photo indexing uses bounded local
derivatives. Apple’s iCloud is contacted only when you explicitly open or
export an iCloud-only original.

## Updates Are Manual for Alpha Builds

There is no auto-update. New builds arrive as new DMGs; install by replacing
the app in `/Applications`.

## Hardware

Apple Silicon and macOS 14 or later. 16 GB memory is recommended.

## Known Limitations

- First model warm takes roughly 10–30 seconds.
- Initial indexing and photo enrichment take time on large collections.
- OCR is English-centric; DOCX and XLSX are not yet indexed.
- Mixed document/photo actions are deliberately bounded and may report
  insufficient support instead of producing an ungrounded answer.
- Live answer streaming is off in this build.
